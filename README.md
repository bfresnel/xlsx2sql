# xlsx2sql

## Goal

This tool aims to easily create sql script from an xlsx file.

## Requirements

- Java JDK 11+
- Apache Maven 3.6.3

## Compiling the project

1. Clone the project

    ```sh
    $ git clone https://github.com/bfresnel/xlsx2sql.git
    ```
    
2. Go to the project folder then use maven

    ```sh
    $ cd xlsx2sql
    ```
    ```sh
    $ mvn clean install
    ```
